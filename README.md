# Os 12 Grandes do Brasil

## Sobre
O Guia Interativo de Clubes Brasileiros é um projeto prático desenvolvido inteiramente com HTML, CSS e JavaScript, o projeto integra duas APIs externas que alimentam o front-end dinamicamente, garantindo que as informações de elenco e história estejam sempre atualizadas.. O resultado é uma página onde você consegue explorar dados históricos e elencos de 12 times do Brasil de forma fluida.
